# Angular Material Summary

## Typography

_customize func arguments (prefix `mat-`)

* `display-1`, 2, 3, 4 (큰 순서로)
* `headline` = h1
* `title` = h2
* `subheading-2` = h3
* `subheading-1` = h4
* `body-1` = 본문
* `body-2` = 본문 bold
* `caption` = 작은 본문체
* `button` = 버튼, 링크용
* `input` = 폼 인풋 영역

Spoqa Han Sans Weights

* thin: 200
* light: 300 
* regular: 400
* bold: 700
